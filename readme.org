* Getting start
** Introduction
This is the configuration file for =feroda 43= distro. And I am using the =i3= version.
I am using the =greetd, tuigreet= to manage the startup of =niri= and =i3=.
I am using the =chezmoi= to manage dotfiles.
 
** Installation
Install the basic packages for applying my dotfiles.
#+begin_src shell
sudo dnf update
# install packages for first use
sudo dnf install -y \
         curl \
         wget \
         git \
         chezmoi
#+end_src

Chezmoi initialize.
#+begin_src shell
chezmoi init https://github.com/MaZhengnan/chezmoi-dot.git
#+end_src

Install other packages.
#+begin_src shell
sudo dnf update
# install packages for first use
sudo dnf install -y \
     # general tools
     libnotify blueman pavucontrol brightnessctl network-manager-applet dnsmasq\
     # file manager
     thunar \ 
     # i3wm related
     conky dunst picom polybar rofi\
     # niri related
     niri waybar sway-lock swaybg fuzzel mako xwayland-satellite
     xdg-desktop-portal-gnome polkit-gnome \
     # shell
     fish kitty \
     # fonts
     google-noto-cjk-fonts wqy-microhei-fonts \
     # editor
     emacs neovim \
     # input method
     fcitx5 fcitx5-qt fcitx5-gtk3 fcitx5-rime 
     
#+end_src

Apply dotfiles.
#+begin_src shell
chezmoi apply
#+end_src

Git initialize. Then I can use git ssh to manage the chezmoi repository.
#+begin_src shell
ssh-keygen -t ed25519 -f ~/.ssh/github
ssh-keygen -t ed25519 -f ~/.ssh/gitee
# Copy the key into github and gitee
ssh -T -v git@github.com
ssh -T -v git@gitee.com
#+end_src

** Packages
*General packages:*
=libnotify=: An application library for sending desktop notifications.
=blueman=: Bluetooth manager. 
=pavucontrol=: Full name is 'Pulse Audio Volume Control', a sample PulseAudio mixer. 
=brightnessctl=: A powerful command-line tool designed to help users easily read and adjust the brightness of their devices.
=network-manager-applet=: A GTK‚Äêbased GUI applet to monitor network status and devices and to start and stop network connections managed by NetworkManager. 
=dnsmasq=: A small and convenient tool for configuring DNS and DHCP.
=thunar=: File manager(Both i3wm and niri).

*I3wm*:
=conky=: System monitoring software for X11 Window System.
=dunst=: Linux desktop notification daemon for X11 or xWayland.
=picom=: A lightweight compositing manager that primarily provides smooth animations and semi-transparency effects for the X11 windowing system.
=polybar=: A tool that allows to create status bars easily and quickly.
=rofi=: Rofi is a window switcher, run dialog, SSH launcher, and an alternative to dmenu. 

*Niri:*
=niri=: Wayland window manager.
=waybar=: Like polybar on wayland.
=sway-lock=: Screen lock. TODO: Don't has clock and background, fix it!
=swaybg=: Setup desktop background picture.
=fuzzel=: Like rofi on wayland. Another one is wofi.
=mako=: Like dunst on wayland.
=xwayland-satellite=: Running x11 programs.
=xdg-desktop-portal-gnome=: The desktop portal recommended by Niri, providing features such as file selection and screen sharing.
=polkit-gnome=: Other software will request permissions through this software.

*Shell:*
